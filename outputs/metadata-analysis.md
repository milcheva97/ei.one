# EI.one Metadata and Page Recreation Notes

Source reviewed: https://ei.one/en/index

## Observed Page Signals

- Page title exposed by the live page: `EI.ONE - The Global Sustainability Ecosystem`.
- Primary H1: `EI - Ecosystem Intelligence`.
- Main keyword cluster: sustainability ecosystem, economy, education, environment, business club, academy, environmental projects, measurable impact, ecosystem intelligence.
- Page intent: introduce EI.one as a global sustainability ecosystem and route users toward discovery, partnership, login, academy, business club, and contact flows.
- Content architecture: sticky navigation, multilingual links, hero with login module, sponsor strip, three core dimensions, origin/vision section, business model, audience segment, calls to action, newsletter, footer, cookie notice.

## Metadata Strategy Used In The Recreation

- Added a focused title under 60 characters: `EI.ONE - Global Sustainability Ecosystem`.
- Added a search-oriented meta description that includes the three-positioning pillars: economy, education, environment.
- Added canonical URL pointing to the original English page.
- Added Open Graph tags for share previews: type, title, description, URL, and large image.
- Added Twitter card tags using the same visual and positioning message.
- Added Organization JSON-LD with name, URL, summary, Swiss address, and phone number from the footer.
- Preserved language declaration with `lang="en"` and responsive viewport metadata.

- The recreation keeps the same structure and core message, while using refreshed, paraphrased copy.
